Marcação de texto

Para marcar um texto basta usar a tag MARK.
Para mudar a cor da marcação basta colocar STYLE e selecionar a cor.

Caso você queira mudar a cor de todas as marcações basta ir no cabeçalho e colocar STYLE, logo em seguida MARK e depopis alterar a cor do background.

Texto grande e Pequeno

Para colocar um texto grande pode ser utilizada a tag BIG, porem esta tag está obsoleta, ou seja ela não é utilizada mais, a forma correta de deixar um texto grande seria com CSS.

Para deixar um texto menor basta usar a tag SMALL, esta tag não está obsoleta portanto pode ser utilizada.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Para colocar o texto como deletado basta usar a Tag DEL.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adcionado depois.

Para colocar ele com o sublinhado basta usar a tag INS.

Também um texto pode ser sublinhado com a tag U porém ela é não semântica e a qualquer momento pode parar de ser considerada.
